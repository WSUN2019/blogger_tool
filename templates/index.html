One-time Google API setup

You only need to do this once. After that, the token is stored locally.

  1. 1 Go to console.cloud.google.com → create a project → search Blogger API v3 and enable it.
  2. 2 Under Credentials, create OAuth 2.0 Client ID (type: Web application). Add http://localhost:5000/blogger/callback as an authorized redirect URI.
  3. 3 Download the JSON file and save it as config/credentials.json inside the blogger_tool folder.
  4. 4 Click Connect with Google below. You'll be redirected to Google to authorize, then returned here.

Blog Management Tools

Blogger Tools

A local web app for formatting, styling, and managing blog posts on Blogger.com — built for speed and for the way you actually write.

Created April 18, 2026 Built with VS Code & Claude Code Python · Flask · Google APIs
Blog Formatter

Paste plain text or light markdown and get fully styled, inline-CSS Blogger-ready HTML. Choose from 7 color themes. Handles sections, lists, callouts, tables, and image grids automatically.

How to use
  1. Paste your text into the input box
  2. Click Smart Format (AI) to let Gemini structure it, or Format Post for manual markdown
  3. Pick a color theme from the row of swatches
  4. Copy the HTML and paste into Blogger's HTML editor
Image Cleaner

Paste any Blogger post HTML to extract every <a><img></a> image block — stripped clean and ready to copy back in. Shows a thumbnail grid so you can see what was found.

How to use
  1. Paste Blogger HTML into the input box
  2. Click Extract Images
  3. Review the thumbnail preview grid
  4. Click Copy All HTML and paste back into Blogger
Blogger Connect

OAuth connection to your Blogger account. Browse posts by year, load HTML, reformat with AI, extract images, and push changes back — without ever leaving this app.

How to use
  1. Set up OAuth credentials in Google Cloud Console (one-time)
  2. Click Connect with Google and authorize
  3. Enter your blog URL to load posts
  4. Filter by year, pick a post, then use ⚡ Reformat + Images
  5. Review the preview and click ↑ Save to Blogger
Smart Format (AI)

Powered by Google Gemini. Paste raw text and the AI auto-detects sections, generates KPI stats, structures lists and tables, and produces clean markdown ready to render. Switch models in ⚙ if you hit a daily quota limit.

Setup
  1. Get a free key at aistudio.google.com
  2. Click ⚙ in Blog Formatter and paste the key
  3. Pick a model — Gemini 2.5 Flash is the default
  4. Click Smart Format (AI) — review the markdown and re-render if needed
